# Sprite Prompt Guide — The Matrix AI Visual

> Guia de prompts padronizados para gerar sprites pixel art dos agentes LMAS.
> Use no ChatGPT (DALL-E) ou qualquer gerador de imagem.

---

## Padrão Visual (OBRIGATÓRIO em todo prompt)

Todo prompt DEVE incluir este bloco base:

```
Pixel art character in Broforce game style. Top-down RPG perspective, slightly angled front view.
Black background (#0D0208). Character stands on transparent ground.
Style: chunky pixel art, visible individual pixels, 2-3px outline, limited palette (max 6 colors per character).
Proportions: large head (~35% of body), compact torso, short legs. Character canvas 32x48 pixels.
Mood: professional, focused, digital-green Matrix aesthetic.
NO text, NO labels, NO UI elements. Character only, centered.
```

## Como Usar

1. **Copie o BLOCO BASE** do início
2. **Cole o prompt do personagem** logo abaixo
3. **Gere no ChatGPT/DALL-E** ou outro gerador
4. **Se precisar ajustar:** adicione instruções específicas ao final
5. **Salve a imagem gerada** em `apps/visual/references/` com nome `{agentId}-reference.png`
6. **Use como referência** para criar o sprite 16x32 no Aseprite

## Notas de Consistência

- Todos usam o mesmo fundo preto (#0D0208)
- Todos têm o mesmo estilo pixel art Broforce (chunky, visible pixels)
- Todos têm proporção cabeça grande (~35% do corpo)
- A cor verde Matrix (#00FF41) aparece em TODOS (hologramas, reflexos, etc)
- Cada personagem tem um ITEM ICÔNICO que o identifica instantaneamente
- A POSE reflete a FUNÇÃO LMAS, não cena de ação do filme
