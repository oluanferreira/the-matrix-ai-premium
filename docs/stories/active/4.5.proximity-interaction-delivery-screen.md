## Story

**As a** usuário (Morpheus/player),
**I want** que ao me aproximar de um agente apareça um balão "..." clicável, e ao clicar (ou clicar direto de longe) abra uma tela grande de interação mostrando a entrega do agente com opções de aprovar/recusar/digitar,
**so that** eu possa interagir naturalmente com os agentes, ver suas entregas em detalhes e dar feedback diretamente no game.

## Acceptance Criteria

1. [x] Quando Morpheus chega perto de um agente (distância < 48px), aparece balão "..." sobre o agente
2. [x] Clicar no balão "..." abre a tela de interação (DeliveryScreen)
3. [x] Clicar direto no agente de longe também abre a DeliveryScreen
4. [x] DeliveryScreen mostra: nome do agente, entrega atual em detalhes (texto grande, scrollável), chat com o agente
5. [x] Botões de ação: [Aprovar], [Recusar], [Digitar] (feedback textual)
6. [ ] Quando agente termina um trabalho, ele se desloca até o Morpheus (pathfinding) — *deferred to v2.0*
7. [ ] Ao chegar, fala algo como "Chefe, vê se está como você me solicitou" e a DeliveryScreen abre automaticamente — *deferred to v2.0*
8. [x] DeliveryScreen é grande o suficiente para mostrar a entrega com detalhes (ex: copy completa, código, etc.)
9. [x] ESC ou botão [X] fecha a DeliveryScreen

## Dev Notes

### Fluxo de Interação

```
1. Morpheus caminha pelo escritório
2. Ao chegar perto de um agente → balão "..." aparece
3. Click no "..." OU click direto no agente → DeliveryScreen abre
4. Na DeliveryScreen: vê entrega, aprova/recusa/digita feedback
5. Fecha com [X] ou ESC

Fluxo alternativo (agente vem até o player):
1. Agente termina trabalho (SSE event: agent-delivery)
2. Agente caminha até o Morpheus (pathfinding)
3. Ao chegar: fala "Chefe, vê se está como você me solicitou"
4. DeliveryScreen abre automaticamente
```

### DeliveryScreen vs InspectorPanel

- InspectorPanel (Story 4.1) = resumo rápido (status, task, última fala)
- DeliveryScreen (Story 4.5) = tela completa de interação (entrega detalhada, chat, ações)
- O click no agente agora abre a DeliveryScreen (InspectorPanel pode ser acessado via ColonistBar)

### Dependências

- Requer Story 4.1 (InspectorPanel como padrão de DOM overlay)
- Requer Story 3.3 (SSE events para detectar entregas)
- Requer PlayerController (para saber posição do Morpheus)
- Requer Story 5.1 parcialmente (pathfinding para agent seek-player)
