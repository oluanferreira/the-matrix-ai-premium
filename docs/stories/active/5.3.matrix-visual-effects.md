## Dev Notes

### Performance

Todos os efeitos devem manter 30+ FPS com 19 sprites. Usar:
- Object pooling para partículas
- Culling para code rain fora da viewport
- Shaders simples (evitar post-processing pesado)

### Dependências

- Requer Story 1.3 (tilemap com janelas)
- Requer Story 5.1 (Smith verdict triggers)
