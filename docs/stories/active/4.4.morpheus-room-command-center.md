## Dev Notes

### Morpheus Room = @lmas-master Terminal

No v1.0, o Morpheus Room é read-only — mostra dados do sistema. No v2.0, será o terminal interativo completo equivalente ao `@lmas-master` na CLI, permitindo executar comandos, criar workflows, gerenciar agentes.

### Dados

Consome do StateManager (Story 3.1/3.2/3.3):
- AgentState[] para lista de agentes
- StoryState para progresso da story
- Últimos handoffs para atividade recente

### Dependências

- Requer Story 1.3 (tilemap — porta no corredor)
- Requer Story 3.3 (SSE — dados em tempo real)
- Requer Story 4.1 (padrão de DOM overlay já estabelecido)
- Opcional: Story 5.3 (efeito glow na porta)
